	*   *

	**
	* *
	*  *
	*   *
	*    *

###Задача 2.

- int sum(int arr[], int n), която намира сумата на числата в масив с n елемента.
- void read(int arr[], int size) - прочита елементите от клавиатурата
- void print(int arr[], int size) - извежда елементите
- bool member(int candidate, int arr[], int size) - проверява дали числото candidate е в масива
