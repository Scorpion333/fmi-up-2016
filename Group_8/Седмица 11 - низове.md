#Нетривиални низове

###Задача 0.
Напишете следните void функции за низове:

- add_to_end (char str[], char sym), която добавя символ в края на низ.
- insert (char str[], char sym, int index), която добавя символ на поределена пизиция в низ (никой символ не трябва да се загуби -
старите символи от позицията нататък трябва да се преместят с 1 място и дължината на низа да се увеличи с 1).
- remove_one (char str[], int index) - премахва символа на позиция index, а останалите символи докрая се преместват "назад" и
дължината се намалява с 1.
- remove_many (char str[], int indexes[], int indexes_size) - премахва символите от съответните индекси
- concat (char a[], char b[]) - записва символите на b в края на a.

В следващите задачи ще считаме, че всички низове са с дължина < 20.

слагайки точки между тях. За целта използвайте двумерен масив char words[8][20] и във всеки ред записвайте по една дума.

	Пример:     C++  Python  Ruby  Java  Perl  JavaScript  Haskell  C#
	Резултат:   C++.Python.Ruby.Java.Perl.JavaScript.Haskell.C#
