#Дефиниция
Дефиниция 0: Символният низ е низ от символи. :)

Дефиниция 1: Символният низ, или стринг, представлява последователност от символи (тип char), които могат да се обработват заедно по някакъв начин.

#Деклариране на символен низ
В нашите задачи ще декларираме низ по следния начин:
char  <име на стринга>[ ]  =  “<съдържание на стринга>”;

Пример:

char  my_string [ ]  =  “I like C++”;

#Достъп да символите на низа
Можем да разберем какъв e, например, осмият символ на  my_string  като напишем my_string[8].
Броим символите от 0 и не пропускаме интервалите, затова осмият символ в горния стринг е '+'.
В квадратните скоби можем да напишем произволно цяло число, което може да е променлива.
Можем дори да променим някой символ.

Пример:

char  account [ ]  =  “Scorpion333”;

cout << account[8] << endl;		                // 3

int a = 4;

cout << account[0] << " " << account[a] << endl; 	// S p

account[9] = '4';

account[10] = account[10] + 2;

cout << account << endl;				// Scorpion345

#Операции с низове
Има много функции, които да приложим върху низове, но засега е достътъчно да знаем, че:

- Можем да използваме cout по начина, по който извеждаме числа и единични символи.
Вече отпечатахме account по този начин

- Можем да разберем дължината на стринг, като напишем: strlen(<име на стрнига>).
За да направим това, трябва да напишем #include<cstring> най-отгоре в програмата.
Дължината представлява броят на символите (включително интервалите)

Пример:
char assistant[ ] = "Peter"

cout << strlen(assistant) + 3 <<  endl;		// 8
